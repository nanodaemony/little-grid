## 1. 需求概述

新增一个功能格子「随机数」，支持生成指定范围内的随机整数。

### 1.1 功能需求
- 设置随机数生成范围（最小值、最大值）
- 设置同时生成的随机数个数（1-100）
- 设置是否允许重复
- 点击按钮生成随机数并展示结果

### 1.2 限制条件
- 仅支持正整数，范围 1 - 999,999,999
- 生成数量上限：100个
- 不保留历史记录

## 4. 数据模型

### 4.1 RandomNumberState

```dart
class RandomNumberState {
  final int minValue;           // 最小值，默认 1
  final int maxValue;           // 最大值，默认 100
  final int count;              // 生成数量，默认 1
  final bool allowDuplicate;    // 允许重复，默认 true
```

## 5. 业务逻辑

### 5.1 输入校验

| 校验项 | 规则 | 错误提示 |
|--------|------|----------|
| minValue | ≥ 1 | "最小值不能小于1" |
| maxValue | ≤ 999,999,999 | "最大值不能超过999999999" |
| minValue < maxValue | 必须成立 | "最小值必须小于最大值" |
| count | 1 - 100 | "数量必须在1-100之间" |
| 不重复时数量限制 | ≤ (max - min + 1) | "范围不足以生成指定数量的不重复随机数" |

### 5.2 生成算法

**允许重复：**
```dart
List.generate(count, (_) => min + random.nextInt(max - min + 1))
```

**不允许重复（Fisher-Yates 洗牌）：**
```dart
final range = List.generate(max - min + 1, (i) => min + i);
for (int i = range.length - 1; i > 0; i--) {
  final j = random.nextInt(i + 1);
  final temp = range[i];
  range[i] = range[j];
  range[j] = temp;
}
return range.take(count).toList();
```

- 数字使用 `headlineLarge` 样式
- 多个数字使用 `Wrap` 布局，居中对齐
- 间距：水平 16px，垂直 16px

## 6. 错误处理

### 6.1 输入错误
- 输入框下方显示红色错误提示
- 有错误时禁用生成按钮
- 失焦时触发校验

### 6.2 边界情况
- 最小值自动修正为不小于1
- 最大值自动修正为不超过999999999
- 当最大值 < 最小值 + 1 时，自动调整
- 不允许重复时，自动限制最大数量为可用范围
